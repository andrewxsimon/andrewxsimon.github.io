Part 2.1: Image "Sharpening"

The unsharp mask filter that we are using is a controlled high-pass filter, built from a blur filter, that emphasizes high frequencies to make the image appear sharper. Blurring the image only gives you low frequencies and thus we can get the high frequencies by subtracting this blurred filter from the image. We then get a sharper image by adding those high frequencies back to the original image.
